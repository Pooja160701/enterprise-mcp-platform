# Roadmap

This roadmap outlines the planned direction for the **Enterprise MCP Platform**. It is intended to provide visibility into the project's evolution and help contributors understand current priorities.

> **Note:** Features and timelines are subject to change based on project goals, community feedback, and available development time.

# Long-Term Vision

The long-term goals for the project include:

- A modular AI orchestration platform built around the Model Context Protocol (MCP)
- Seamless integration with a wide range of MCP-compatible servers
- A secure and extensible plugin ecosystem
- Production-ready deployment patterns
- Strong observability and operational tooling
- A contributor-friendly open-source architecture

# Release Strategy

The project follows Semantic Versioning.

| Version | Meaning |
|----------|---------|
| Major (`2.0.0`) | Breaking changes or significant architectural updates |
| Minor (`1.1.0`) | New features with backward compatibility |
| Patch (`1.0.1`) | Bug fixes, documentation updates, and minor improvements |
